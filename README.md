# Blinkit Sales Anlysis dashboard built using Microsoft Power Bi the dashboard Analyzes grocery sales data to identify key trends, product demand, and outlet performance.

## Project Overview
This project is an interactive sales analysis dashboard built using Microsoft Power BI. The dashboard analyzes grocery sales data to identify key trends, product demand, and outlet performance.

## Tools Used
- Microsoft Power BI
- Microsoft Excel

## Dashboard Features
- Total Sales KPI
- Average Sales KPI
- Number of Items Sold
- Average Rating

## Visualizations
- Donut Chart – Fat Content Analysis
- Bar Chart – Sales by Item Type
- Column Chart – Outlet Location Analysis
- Line Chart – Outlet Establishment Trend
- Pie Chart – Outlet Size Distribution

## Key Insights
- Total sales reached approximately $1.2M indicating strong demand.
- Fruits and vegetables contribute the highest share of revenue.
- Medium-sized outlets generate higher sales compared to small outlets.
- Tier-1 and Tier-2 locations show stronger sales performance.

## Business Recommendations
- Expand high-performing product categories.
- Focus on high-demand outlet locations.
- Increase product variety to attract more customers.
- Improve customer satisfaction and ratings.
